关于 PIME

PIME 输入法平台网站：PIME 输入法平台
PIME 输入法平台版本：


PIME 是在 Windows 上使用 Python 3 开发 TSF 中文输入法的平台，由作者 洪任谕(PCMan) 所开发，目前支援到 Windows 10，并且能支援 Windows app container (Metro mode)。可以让开发者在不了解系统细节的情况下，单纯使用 Python 就能实作支援 Windows 的输入法模组。

「拧码苏哟法 (PIME)」是在 PIME 输入法平台下，基于 CIN→JSON 码表所实现的拧码苏哟法模组。
